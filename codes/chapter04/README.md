# frontend-tutorial - chapter04 示例

## 说明

- **babel**

执行以下命令，可以看到执行结果：

```sh
$ npm install
$ npm start
```

- **es6**

所有脚本可以使用以下命令执行：

```sh
$ node xxx.js
```

- **eslint**

存放 eslint 的配置文件示例。

- **jigsaw**


整合目前学习的知识点。

安装依赖库

```sh
$ npm install
```

启动开发服务器：

```sh
$ npm run dev
```

编译产生静态文件，默认输出目录为 `dist` ：

```sh
$ npm run prod
```

eslint 检查：

```sh
$ npm run lint
```
